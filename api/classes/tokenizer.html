Class Tokenizer

The tokenizer is used to prepare the string content of a document field for the inverted index. Firstly the string gets split into tokens. After that the tokens will be trimmed/stemmed with defined functions from the queue.

  • To change the splitter function, use {@link Tokenizer#setSplitter}.
  • To add functions to the queue, use {@link Tokenizer#add}, {@link Tokenizer#addBefore} and {@link Tokenizer#addAfter}.
  • To remove a function from the queue, use {@link Tokenizer#remove}.
  • To reset the tokenizer, use {@link Tokenizer#reset}.
