Class QueryBuilder

This query builder is the root of each query search. The query contains a sub query and parameters for setup scoring and search options.

Possible sub query types are: TermQueryBuilder, TermsQueryBuilder, FuzzyQueryBuilder, WildcardQueryBuilder, MatchQueryBuilder, MatchAllQueryBuilder, PrefixQueryBuilder, BoolQueryBuilder, ConstantScoreQueryBuilder, ExistsQueryBuilder

example

new QueryBuilder() .finalScoring(true) .useBM25(1.5, 0.5) .term("first_name", "albert") .build(); // The resulting documents: // contains the first name albert // are scored and ranked using BM25 with k1=1.5 and b=0.5
